# Quick Start: Initializing the Project

This document provides the essential command to get the project started.

## Docusaurus Initialization

To create the initial Docusaurus project structure, run the following command in your terminal. This command will set up a new Docusaurus site in a directory named `physical-ai-book` using the "classic" template with TypeScript support.

```bash
npx create-docusaurus@latest physical-ai-book classic --typescript
```

### Command Breakdown
- `npx create-docusaurus@latest`: Executes the latest version of the Docusaurus scaffolding tool.
- `physical-ai-book`: The name of the directory where the project will be created.
- `classic`: The template to use, which includes standard documentation features, a blog, and pages.
- `--typescript`: A flag to configure the project for TypeScript out-of-the-box.
